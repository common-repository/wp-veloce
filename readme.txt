WordPress Veloce increases WordPress page load speed to get a higher score on the speed testing services like Google PageSpeed Insights, Gtmetrix, Pingdom etc.

== Description ==

WordPress Veloce is a wordpress plugin that optimizes your wordpress website improving page loading speed. 

Just 1 click setup and wordpress is optimized with the various best practices to get a higher score on the speed testing services. Increase of page load speed gives you a better performance for loading the website. WordPress Veloce works with all the themes as its not affecting any theme or plugin code. It can be used in combination with other Cache plugin without problems.

Increase your WordPress page load speed to get a higher score on the speed testing services like Google PageSpeed Insights, Gtmetrix, Pingdom etc.


== Installation ==


1. Download the plugin and unzip it.
2. Upload the folder wp-veloce to your /wp-content/plugins/ folder.
3. Activate the plugin through the 'Plugins' menu in WordPress
